This replaces the first 'Defeat Enemy' battle in the training dummy. Note that you cannot switch armour/weapons mid battle.

USAGE: The first character you add will be the one you will be playing. I highly recommend sticking to Jack as other characters are not stable. You also have to add your allies, anyone in your party will not be in the battle unless added.

The first field is the character id, which you can find below. Please keep it to just the value and nothing else (ex. 035 for Kain)

The second field is the character's position. You should start off with 96 for the first character. For allies, continue from 96 (97, 98, 99, 9A, 9B, etc). For enemies, start from A0 and continue onwards (A1, A2, A3, A4, A5, etc).

The third field is the characters apperance. For most cases you can just have this at 0. Some characters might need a different value to function in battle. (Aphelion needs to be 05 for Silver Dragon form). If you are unsure, put 0.

The fourth field is vulnerability status. You will probably just want to have this at 0. If you want to be unkillable/make the enemy unkillable, put 80, otherwise put 0.

The fifth field determines if the character is an ally or enemy. If you want it to be an ally put 0, otherwise put 11 to make them an enemy.

Once you've filled out those five fields you can press add and it will add that character to the battle. You can start the process over adding more to a maximum of 10 total.

The arena and bgm fields determine the battle arena you play in as well as the music that plays. You can leave this blank to have the generic training dummy arena/music. Values for this can be found below.

Once you're done you can hit the generate button and it will download a .pnach file. You put this in your PCSX2 cheats folder. Make sure to enable cheats before starting the game.
